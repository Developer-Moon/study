The dataset is from: https://data.giss.nasa.gov/cgi-bin/gistemp/stdata_show_v4.cgi?id=USW00024233&ds=14&dt=1

The GISS Surface Temperature Analysis ver. 4 (GISTEMP v4) is an estimate of global surface temperature change. This dataset is taken from the SeaTac weather station.

This is a modified version of the original dataset. It has been cleaned up for learning and testing purposes as an example dataset for time series prediction.
